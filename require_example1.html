What is Require.js?

            Require.js is an implementation of Asynchronous Module Definition (AMD). The AMD specifies a mechanism for defining modules such that the module and its dependencies can be asynchronously loaded. This is particularly well suited for the browser environment where synchronous loading of modules incurs performance, usability, debugging, and cross-domain access problems.
            Require.js allows to load JavaScript files dynamically.
            Instead of putting a dozen <‍script> tags at the top of your page, Require.js will load them for you, when you need it.
            Require.js can be configured in detail to know where to find the correct javascript files
            JavaScript files are called modules in the scope of Require.js
            A module used by Require.js contains a function that returns something.
            This something is cached by Require.js. The next time somebody tells Require.js to get the same module, the cached value is returned.
        
What is AMD?
